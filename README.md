# Wiki-clone
This project is a simple Wikipedia-inspired web application built using HTML, CSS, and JavaScript. It replicates the core layout and functionality of a typical encyclopedia website, allowing users to search and browse articles in a clean, structured interface.


📌 Features
🔍 Search Articles – Quickly find topics using the search bar
📄 Dynamic Content Display – Articles load dynamically using JavaScript
🎨 Responsive Design – Works on mobile, tablet, and desktop
⚡ Fast Performance – Built with vanilla JavaScript (no frameworks)
🧭 Simple Navigation – Easy-to-use layout inspired by Wikipedia
🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & responsiveness
JavaScript (ES6) – Logic & interactivity
